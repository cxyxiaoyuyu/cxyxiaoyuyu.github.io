发布自己的npm包

Posted on 2020-07-19 Edited on 2020-07-20 In node

创建npm项目

1 创建一个目录 进入这个目录
2 npm init 创建 package.json

name 包名
version 版本号 通常是1.0.0
main 入口文件 通常是index.js

3 创建入口文件 index.js
4 入口文件 index.js 添加如下内容

// 导出 printMsg函数
exports.printMsg = function() {
  console.log("This is a message from the demo package");
}

5 创建readme.md 文件 发布后会显示在页面上

发布到npm官网

1 注册npm账号
2 npm login 登录 npm whoami 确认登录
3 npm publish 发布到官网
4 测试 https://npmjs.com/package/ 查看是否发布成功

使用自己的npm包

1 进入另一个目录
2 npm install
3 创建一个test.js文件，requier这个包，并调用printMsg方法

let value = require('xiaoyu-npm-demo')
value.printMsg()

4 执行node test.js 查看是否有输出
