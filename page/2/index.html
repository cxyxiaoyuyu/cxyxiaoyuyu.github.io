什么是深拷贝？

简单理解: b 是 a 的一份拷贝，b 中没有对 a 中对象的引用
另一种理解: b 是 a 的一份拷贝，a、b 各自画图，a 与 b 没有连接

  • 不支持函数

  • 不支持日期(会把 date 变成 iso8601 格式的日期字符串)
  • 不支持正则
  • 不支持 Error 对象
  • 不支持环状结构
  • NaN Infinity 会变成 null
